## 🚀 Quick Start

1. Download the latest release from [**Releases**](https://github.com/dexifyink/MOONLIGHT/releases)
2. Open CS2 in **Fullscreen Windowed** mode
3. Run `MOONLIGHT.exe`
4. Press **INSERT** to toggle the menu
5. Press **END** to exit

> [!IMPORTANT]
> Make sure your game is running in **Fullscreen Windowed** mode ❗
