## 🚀 Quick Start

1. Download the latest release from [**Releases**](https://github.com/dexifyink/MOONLIGHT/releases)
2. Open CS2 in **Fullscreen Windowed** mode
3. Run `MOONLIGHT.exe`
4. Press **INSERT** to toggle the menu
5. Press **END** to exit

> [!IMPORTANT]
> Make sure your game is running in **Fullscreen Windowed** mode ❗

## 📘 Developer Instructions

1. **Clone** the repository with dependencies:

```sh
git clone --recursive https://github.com/dexifyink/MOONLIGHT
```

> If you cloned before submodules were added: `git submodule update --init --recursive`

2. **Build** using **Visual Studio 2022** (or later) — Configuration: **`x64 - Release`**

3. **Locate** your binary in `<arch>/<configuration>`, e.g., `x64/Release`
